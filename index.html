История создания HTML

1989 - Начало

Тим Бернерс-Ли, работая в CERN (Европейская организация по ядерным исследованиям), предложил систему для обмена документами между учёными. Так появилась концепция Всемирной паутины (World Wide Web).

1991 - Первая версия

Опубликована первая публичная версия HTML с 18 тегами. Это был простой язык разметки, позволяющий создавать гипертекстовые документы.

1995 - HTML 2.0

Первый официальный стандарт HTML. Появились формы, таблицы и возможность загрузки файлов.

1997 - HTML 3.2 и 4.0

HTML 3.2 стал первым стандартом W3C. HTML 4.0 принёс поддержку CSS, скриптов и улучшенную доступность.

2014 - HTML5

Современная версия с семантическими тегами, мультимедиа (video, audio), Canvas для графики, локальным хранилищем и многим другим.

2023 - HTML Living Standard

HTML теперь развивается как "живой стандарт" WHATWG, постоянно обновляется и улучшается.

Создатель HTML

Сэр Тимоти Джон Бернерс-Ли

Родился: 8 июня 1955 года
Национальность: Британец
Образование: Оксфордский университет

Британский учёный-информатик, изобретатель World Wide Web, HTML, URL и HTTP. Основатель и директор Консорциума Всемирной паутины (W3C).

В 1989 году он предложил систему управления информацией на основе гипертекста, которая позже стала Всемирной паутиной. За свои достижения был посвящён в рыцари королевой Елизаветой II в 2004 году.

Создатели популярных плагинов и библиотек

jQuery

Создатель: John Resig

Библиотека JavaScript, выпущенная в 2006 году. Упрощает работу с DOM, событиями, анимациями и AJAX. Девиз: "Write less, do more".

JavaScript 2006

React

Создатель: Jordan Walke (Facebook)

Библиотека для создания пользовательских интерфейсов, выпущена в 2013. Использует виртуальный DOM и компонентный подход. Одна из самых популярных библиотек в мире.

Frontend 2013

Three.js

Создатель: Mr.doob (Ricardo Cabello)

Библиотека для создания 3D-графики в браузере на основе WebGL. Позволяет создавать потрясающие 3D-сцены, игры и визуализации.

3D Graphics 2010

Bootstrap

Создатели: Mark Otto & Jacob Thornton

Популярнейший CSS-фреймворк от Twitter (2011). Предоставляет готовую сетку, компоненты и утилиты для быстрой разработки адаптивных сайтов.

CSS Framework 2011

Phaser

Создатель: Richard Davey

Фреймворк для создания 2D-игр на HTML5 и JavaScript. Поддерживает Canvas и WebGL. Идеален для браузерных игр.

Game Dev 2013

Chart.js

Создатель: Nick Downie

Простая библиотека для создания красивых анимированных графиков и диаграмм на элементе Canvas. Поддерживает 8 типов диаграмм.

Charts 2013

D3.js

Создатель: Mike Bostock

Библиотека для визуализации данных с помощью SVG, Canvas и HTML. Позволяет создавать сложные интерактивные визуализации данных.

Data Viz 2011

Socket.io

Создатель: Guillermo Rauch

Библиотека для работы с WebSockets. Обеспечивает двустороннюю связь между клиентом и сервером в реальном времени. Идеальна для чатов и игр.

Real-time 2010

Lodash

Создатель: John-David Dalton

Утилитарная библиотека JavaScript, предоставляющая функции для работы с массивами, объектами, строками и числами. Улучшенная версия Underscore.js.

Utility 2012

Leaflet

Лёгкая open-source библиотека для интерактивных карт. Поддерживает различные тайловые слои, маркеры, полигоны и многое другое.
